Project: Toxicity Classification

Dataset:
Jigsaw Toxic Comment Dataset

Models:
1. Logistic Regression
2. Linear SVM

Results:

Logistic Regression:
Accuracy = 95.77%
Precision = 0.91
Recall = 0.62
F1 = 0.74

Linear SVM:
Accuracy = 95.99%
Precision = 0.88
Recall = 0.68
F1 = 0.76

Conclusion:
Linear SVM performed better overall, especially in detecting toxic comments.
